# How to create script to run firefox with custom profile
- Open Script Editor (use Spotlight or folder Applications/Utilities
- Insert code from the firefox_user.applescript file
- Change User1 on exact profile name
- Select File->Save
- - Format: Application
- - Name: Firefox - user (or something like that)
- - Location: Applications or Decktop

# To change icon:
- make a right click on firefox_user.app
- select "Get Info"
- copy preferred icon and place it on the upper left corner of the "Get Info" window
- to hide native Firefox icon, run firefox_hide_icon.sh file and restart firefox.
